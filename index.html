Git and Git Hub

  1. Git Hub hosting website designed for programming projects using the Git version control system. It was created using the Ruby on Rails framework and the Erlang language. The website has been operating since April 2008. GitHub provides free hosting for open source programs and private repositories (some private repository features are paid). In June 2018, it was announced that the website would be acquired by Microsoft for $ 7.5 billion.
  2. W maju 2019 roku GitHub informuje, że ma około 37 milionów użytkowników i więcej niż 100 milionów repozytoriów (w tym co najmniej 28 milionów repozytoriów publicznych).
